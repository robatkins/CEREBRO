# Cerebro

A local AI agent designed to manage your Python codebase autonomously.

## Overview

Cerebro is an intelligent agent that operates within your workspace directory to:
- Reason about your code and project structure
- Edit files safely with evidence-based decisions
- Coordinate specialist sub-agents for bounded work
- Run autonomous improvement loops when requested
- Provide concrete results after analysis or tasks

## Features

### Core Capabilities
- Local Operation: Runs entirely on your computer, no external API required
- Workspace Confinement: Operates only within the designated workspace directory
- Role-Based Sub-Agents: Delegates specialized work to role-based sub-agents (planner, coder, reviewer)
- Autonomous Improvement Loops: Can run self-improvement cycles with configurable cycle limits
- Memory System: Persistent memory for tasks and facts across sessions
- Control Modes: Supports continue/wrap_up/stop modes for autonomous loops

### Tools Available
Cerebro provides a comprehensive set of tools including:
- File operations (read, write, search)
- Python validation and compilation checks
- Git integration (status, diff, log, branch)
- Team execution templates
- Quality gates and meta-review processes
- Memory management for tasks and facts

## Installation

No external dependencies required. Cerebro runs as a standalone Python agent.

### Prerequisites
- Python 3.x environment
- Git (optional, for version control integration)

## Usage

### Basic Operation
```bash
# Start the agent in your workspace
cd /path/to/workspace
python -m cerebro.agent --workspace .
```

### Configuration
Cerebro reads configuration from:
- `config.py`: Main configuration file
- `.agent_checkpoints/`: Persistent state and checkpoints
- Control file: Manages autonomous loop modes

### Team Execution
For complex tasks, Cerebro can execute teams of roles:
```python
# Example team execution
run_team(task="improve code quality", roles=["planner", "coder", "reviewer"])
```
